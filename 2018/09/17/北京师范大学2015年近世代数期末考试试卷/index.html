北京师范大学2015年近世代数期末考试试卷

发表于 2018-09-17 | 分类于 学习
近世代数 期末 刘玉明.

试题

  1. (20分) \(\mathbb{Z}_n=\{[0],[1],\cdots,[n-1]\}\) 表示模 \(n\) 的剩余类加群.
    1. 写出 \(\mathbb{Z}_4\) 的加法运算表.
    2. 写出 \(\mathbb{Z}_6\) 的所有子群.
    3. \(\mathbb{Z}_7\) 有几个子群?
    4. \(\mathbb{Z}_{28}\) 有几个 \(7\) 阶元?
  2. (20分) \(\mathbb{Z}_n^*=\{[a]\in\mathbb{Z}_n~|~a~与~n~互素\}\).
    1. 证明: \(\mathbb{Z}_n^*\) 关于乘法 \([a][b]=[ab]\) 构成一个阿贝尔群.
    2. 求 \([3]\) 在 \(\mathbb{Z}_8^*\) 以及 \([5]\) 在 \(\mathbb{Z}_{13}^*\) 中的逆元.
  3. (15分) 用 \(S_5\) 表示 \(5\) 次对称群.
    1. \(\sigma=(12345)\) 是 \(S_5\) 中的几阶元?
    2. \(S_5\) 中与 \(\sigma\) 共轭的元素有几个?
    3. 写出循环群 \(\langle\sigma\rangle\) 中的所有元素. \(\langle\sigma\rangle\) 是不是 \(S_5\) 的正规子群?
  4. (20分)
    1. 写出整数环 \(\mathbb{Z}\) 的所有极大理想.
    2. 写出模 \(18\) 的剩余类环 \(\mathbb{Z}_{18}\) 的所有极大理想.
    3. \(x^2+1\) 是否是 \(\mathbb{Z}_3[x]\) 中的不可约多项式?
    4. \(\mathbb{Z}_3[x]/\langle x^2+1\rangle\) 是怎样的环?
  5. (15分) 用 \(R\) 表示环 \(\mathbb{Z}[i]/\langle 1+3i \rangle\), 其中 \(\mathbb{Z}[i]\) 表示高斯整环, 证明:
    1. 在 \(R\) 中, \([i]=[3]\), \([10]=[0]\), \([a+bi]=[a+3b]\), \(\forall a, b\in\mathbb{Z}\).
    2. \(\varphi:\mathbb{Z}\to R~(a\mapsto [a])\) 是环的满同态.
    3. \(\ker\varphi=10\mathbb{Z}\). 从而 \(R=\mathbb{Z}_{10}\).
  6. (10分)
    1. 设 \(E=F(\alpha)\) 是域 \(F\) 的单超越扩域, \(\beta\in E\backslash F\). 证明: \(\alpha\) 是 \(F(\beta)\) 上的代数元.
    2. 证明或否定一下命题: 设 \(E/F\) 是域扩张, \(\alpha\in E\), \(m,n\in\mathbb{Z}^+\). 如果 \(\alpha\) 是 \(F\) 上 \(n\) 次代数元, 且 \(m\) 与 \(n\) 互素, 那么 \(\alpha^m\) 也是 \(F\) 上的 \(n\) 次代数元, 且 \(F(\alpha^m)=F(\alpha)\).

懒！没有解答！
